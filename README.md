## Run Locally

Clone the project

```bash
  git clone https://github.com/anthonycory/ReactNative-MangaApp
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  expo start
```
